# LOL Explorer

Anroid application, demonstrating [RIOT League Of Legends API][RIOT] usage
---

![main][Main]

##### Supported API calls:
* champion-v1.2
* current-game-v1.0
* game-v1.3
* league-v2.5
* lol-static-data-v1.2
* match-v2.2
* stats-v1.3
* summoner-v1.4
* team-v2.4

##### Used libraries
[Cupboard][Cupboard] - as ORM 

[Retrofit][Retrofit] - library for working with REST API
